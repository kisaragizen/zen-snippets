## 1. gakmas-autoclick.ahk（学マス连点器）
需要安装**AutoHotkey**，且注意脚本运行时会自动申请UAC权限，**必须允许**，否则无法在游戏界面生效。

**功能**（所有操作都有蜂鸣或文字提示）：  
- 右键单击 → 开启/关闭连点（方便有时你想自己点）  
- 右键长按 → 暂停/恢复脚本（方便你把游戏挂后台去做其他事情）  
- 连点开启时，可以通过鼠标滚轮调节点击频率 → 通过调节点击间隔±50ms（每秒最快点20次）  
- 左键右键同时长按 → 退出脚本

## 2. kf-message-repair.js（KF论坛私信截断BUG修复）
Tampermonkey脚本，修复绯月论坛私信功能中因“引用标签强制截断图片标签”导致的私信内容无法正常显示的BUG。  
可以直接在[GreasyFork中安装](https://greasyfork.org/zh-CN/scripts/536905)，或在Tampermonkey中新建脚本并复制脚本代码。
